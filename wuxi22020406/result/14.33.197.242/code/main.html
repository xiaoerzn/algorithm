What is UMS library plugin?

UMS library plugin has to be installed to use UMS-WebViewer on Chrome.

There is the only version to UMS library plugin:

  • New version which can be used after downloading and is developed specially for Windows XP and over.

This instructions is an explanation about How to confirm an installation of UMS library plugin for Chrome and How to install on Windows XP and over.
Many Windows users use UMS-WebViewer on IE but we would like to recommend to install a new plugin for others who use on Chrome.


Please check if plugin is installed.

The plugin is not installed if you can see this message.


How to install a plugin

  1. Click here to download a plugin.
  2. Save the file on Desktop.
  3. Install the Plugin as running the file when the download is completed.
  4. Close a Chrome after the installation is completed.
  5. Start the Chrome again.
